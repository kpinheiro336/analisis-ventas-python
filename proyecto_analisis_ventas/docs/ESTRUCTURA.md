# Estructura del repositorio

## Archivos principales

- `Analisis_Ventas.py`
  - Archivo principal del proyecto.
  - Incluye menú interactivo, estadísticas, visualizaciones, normalización y predicción.

- `requirements.txt`
  - Lista de dependencias del proyecto.

- `README.md`
  - Documento principal con la explicación general del repositorio.

## Documentación de apoyo

- `docs/RESUMEN.md`
  - Resumen breve del proyecto orientado a portfolio o CV.

- `docs/ESTRUCTURA.md`
  - Descripción de carpetas y archivos del repositorio.

## Gestión del proyecto

- `CHANGELOG.md`
  - Historial de cambios principales del proyecto.

- `AUTHORS.md`
  - Autoría y contacto.

## Recursos visuales

- `assets/`
  - Carpeta para capturas y recursos visuales del proyecto.
